# 💍 Adil & Nadiya — Wedding Planner

Application web de gestion de mariage pour le **26 juin 2026**.

## Fonctionnalités

### 👥 Gestion des invités
- Ajout, modification et suppression d'invités
- Rôles : témoin, demoiselle d'honneur, garçon d'honneur, famille, enfant...
- Côtés : Adil / Nadiya / Commun
- Suivi RSVP (confirmé, en attente, décliné)
- Régimes alimentaires
- Filtres et recherche

### 🪑 Plan de table
- Création de tables (rondes, rectangulaires, longues)
- Capacité configurable
- Assignation des invités aux tables
- Visualisation des places libres
- Alerte des invités non placés

### 🍽️ Menu
- Sections : Cocktail, Entrées, Plats, Desserts, Boissons
- Badges végétarien / halal
- Description de chaque plat

### 💰 Budget
- Catégories : salle, traiteur, musique, photo, fleurs, tenues, etc.
- Suivi prévu vs payé vs reste à régler
- Graphique par catégorie
- Statuts : prévu, acompte versé, payé

### ✅ Checklist
- Tâches pré-configurées avec dates cibles
- Priorités (haute, moyenne, basse)
- Filtres par statut et priorité
- Progression globale

### 📅 Planning
- Programme détaillé du jour J
- Timeline de tous les événements
- Ajout d'événements personnalisés

### 🏠 Tableau de bord
- Vue d'ensemble avec compteur de jours
- Stats clés (invités, budget, tâches, tables)
- Répartition RSVP
- Prochaine tâche urgente

## Installation

Aucune installation requise. Application 100% front-end (HTML + CSS + JS vanilla).

```bash
# Ouvrir directement dans le navigateur
open index.html

# Ou servir localement
npx serve .
# ou
python3 -m http.server 8000
```

## Données

Les données sont sauvegardées automatiquement dans le **localStorage** du navigateur. Aucun serveur nécessaire.

## Stack

- HTML5 sémantique
- CSS3 custom (variables, grid, flexbox)
- JavaScript vanilla (ES6+)
- Polices : Cormorant Garamond + Jost (Google Fonts)
- Persistance : localStorage
